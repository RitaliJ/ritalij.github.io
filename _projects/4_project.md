Optimized inference for a Qwen2 transformer model on GPU by writing bespoke CUDA kernels for attention, LayerNorm, and rotary embeddings. I tuned kernel launch parameters and memory reuse for efficient autoregressive decoding, achieving roughly 5x faster throughput than a PyTorch baseline and strengthening the system's relevance for robotics-oriented decision-making pipelines. Key technologies: CUDA, transformer inference, and ML systems engineering.
