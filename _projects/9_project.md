Designed and built a compact Hall effect mini piano using a rotational mechanism on a shared axle, a magnet mounted beneath each key, and a Hall effect sensor to detect displacement. As each key is pressed, the magnet moves closer to the sensor and produces a continuous analog voltage proportional to the magnetic field, enabling expressive control such as velocity sensitivity. The system was controlled by an Arduino Nano that read the sensor outputs and transmitted note data to a computer for sound synthesis. Key technologies: embedded systems, Hall effect sensing, and Arduino.
