Designed and implemented a low-latency camera tracking robot with a two-axis gimbal, using PID control and polynomial-spline planning. I integrated OpenCV-based object detection with multithreaded C control logic to steer the camera smoothly, achieving approximately 95% detection accuracy at 60 FPS with about 10 ms latency. Key technologies: robotics, OpenCV, embedded systems, and real-time control.
