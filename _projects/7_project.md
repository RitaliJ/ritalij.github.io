Built a camera gimbal with two rotating motors for autonomous object detection and tracking. The system used continuous motion planning with polynomial splines and trajectory calculations to steer the camera smoothly, while OpenCV-based image processing handled detection and tracking. I also integrated the controller and detector code using multiple threads, improving baseline detection accuracy to roughly 95%. Key technologies: robotics, OpenCV, and embedded systems.
