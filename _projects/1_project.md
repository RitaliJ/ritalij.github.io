Built a real-time hand gesture recognition system using a photodiode array, Arduino-based data acquisition, Python serial communication, and lightweight neural-network inference. I collected and labeled gesture data for four classes — none, peace, C, and T — and exported a TorchScript model for live classification. Key technologies: Embedded Systems, Computer Vision, and PyTorch. Project materials and demo resources are available here: https://drive.google.com/file/d/1eR-0hXjuXWR_mL-TOV9wIbJpDBLqBody/view?usp=sharing.
