Built a real-time hand gesture recognition system using a photodiode array, Arduino-based data acquisition, Python serial communication, and lightweight neural-network inference. I collected and labeled gesture data for four classes — none, peace, C, and T — and exported a TorchScript model for live classification. Placeholder image and demo resources coming soon.
